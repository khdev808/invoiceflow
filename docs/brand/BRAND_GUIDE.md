# InvoiceFlow — Quiet Ledger Brand Guide

## Positioning

InvoiceFlow is the **permanent financial companion** for solo businesses: calm, precise, and always there when money matters.

## Color palette

| Token | Hex | Usage |
|-------|-----|-------|
| Ivory | `#FAF8F4` | Page backgrounds |
| Surface | `#FFFFFF` | Cards, inputs |
| Navy | `#0F1419` | Primary text, logo strokes |
| Muted | `#5C6570` | Secondary text |
| Border | `#E8E4DC` | Dividers, input borders |
| Brass | `#C9A227` | Primary accent, CTAs, active nav |
| Brass dark | `#B8860B` | Hover states |
| Brass soft | `#F5EDD6` | Accent backgrounds |
| Success | `#1B6B4A` | Paid, positive states |

## Typography

- **UI:** Instrument Sans (400, 500, 600)
- **Display:** Fraunces (600) — hero headlines only
- Body line-height: 1.6; headings: tight tracking (-0.02em)

## Shape & motion

- Card radius: 12px; inputs: 8px
- Shadows: warm, low opacity — never colored glows
- Transitions: 200–280ms ease-out
- No emoji in product UI

## Voice

- Calm, confident, never hype
- Stability phrases: "Saved", "Synced", "On record", "Your ledger"
- Avoid: exclamation-heavy CTAs, trash-talk in-app

## Logo

Use the folded-ledger mark + wordmark lockup. Never use the legacy "IF" gradient box.

## Illustrations

Minimal line art: thin navy strokes (#0F1419), brass accent dot (#C9A227), warm ivory fills (#FAF8F4).
